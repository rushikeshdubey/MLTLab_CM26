📖 How to Interpret Confidence Scores

  • Score ≥ 0.7 (70%): High confidence positive sentiment
  • Score 0.5-0.7 (50-70%): Moderate confidence - leaning positive
  • Score = 0.5 (50%): Uncertain/Borderline - may need more context
  • Score 0.3-0.5 (30-50%): Moderate confidence - leaning negative
  • Score ≤ 0.3 (30%): High confidence negative sentiment

Formula: If score ≥ 0.5 → Positive (Confidence = score × 100%)
If score < 0.5 → Negative (Confidence = (1-score) × 100%)
